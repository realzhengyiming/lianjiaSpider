本人准备外出实习，然后就需要租房，然后就想到了查一下网上的租房信息，不知道怎么的就看到链家这个网站，然后顺势就进行一波基本信息的爬取，然后进行一些简单的分析，以方便自己找房子租😁欢迎大家一起讨论租房的东西
